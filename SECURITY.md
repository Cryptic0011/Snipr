# Security Policy

Snipr is local-first: captures, recordings, OCR text, and preferences never
leave your machine. The only network traffic is the Sparkle update check
against the signed appcast on GitHub Pages.

## Reporting a vulnerability

Please report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/Cryptic0011/Snipr/security/advisories/new)
rather than opening a public issue. You'll get a response within a week.

## Supported versions

Only the [latest release](https://github.com/Cryptic0011/Snipr/releases/latest)
receives fixes.
